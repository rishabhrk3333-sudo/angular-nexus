Change Detection

Change Detection is Angular’s mechanism for keeping the UI synchronized with application data.

What is Change Detection?

Change Detection is the mechanism by which Angular checks and updates the DOM whenever application data changes. Angular traverses the component tree, compares bindings, and updates the UI if values changed.

Change Detection Cycle

1️⃣ User/Event Trigger

2️⃣ Angular Starts CD

3️⃣ Components Checked

4️⃣ DOM Updated
