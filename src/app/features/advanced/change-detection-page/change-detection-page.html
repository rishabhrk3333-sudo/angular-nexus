Change Detection

Change Detection is Angular’s mechanism for synchronizing data and the DOM. It ensures the UI updates automatically whenever application state changes.

What is Change Detection?

Angular creates a tree of components and checks each component whenever data changes. It compares previous and current values, then updates the DOM only where needed.

1. Trigger

User event or async task occurs.

2. Traverse

Angular checks component tree.

3. Compare

Previous and current values compared.

4. Update

DOM updates automatically.
